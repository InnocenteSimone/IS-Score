Band and Edges Detection#

The Band-Edge Detection module provides a collection of utility functions that support the core functionality of the IS-Score package.

In particular is the module responsible for detecting the bands and edges in the Raman spectrum, which are essential for calculating the IS-Score.

API Reference#

findBands(sp: array, tolerance: int) → list#

Find the meaningful bands in a Raman spectrum

Parameters:

  • sp (np.array) – The Raman spectrum.

  • tolerance (int) – The tolerance value used to consider two bands as “common bands”.

Returns:

filtered_bands – The list containing the detected band.

Return type:

list

getBandEdges(sp: array, bands: list) → list#

Find the edges for each band in the list.

Parameters:

  • sp (np.array) – The Raman spectrum.

  • bands (list) – The list containing the bands of which edges need to be detected.

Returns:

edges – The list containing the detected edges.

Return type:

list

getWlenProminences(sp, bands, edges)#

Retrieve the bands prominence using the wlen parameters. The wlen parameter is set at twice the distance between the edges of the band.

Parameters:

  • bands (list) – The list containing the bands.

  • edges (list) – The list containing the edges.

Returns:

The list with the prominences.

Return type:

list
